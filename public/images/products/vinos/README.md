## Formato Recomendado

- **Formato:** JPG
- **Dimensiones:** 400x300 píxeles (o proporcional)
- **Calidad:** Alta calidad para mostrar los detalles del producto
- **Peso:** Máximo 200KB por imagen para optimizar la carga

## Fallback

Si una imagen no está disponible, el sistema debe mostrar una imagen placeholder genérica de vinos.
